# 激活码生成工具使用说明

## 📋 快速开始

### 方法一：交互式使用（推荐）

1. 打开终端，进入项目目录：
   ```bash
   cd /Users/jane/Desktop/Invoice-Project
   ```

2. 运行脚本：
   ```bash
   python3 generate_activation_code.py
   ```

3. 输入用户提供的机器码，例如：`A3F2-8B4C-D1E9-7A5B`

4. 获得激活码并发送给用户

---

### 方法二：命令行快速生成

```bash
python3 generate_activation_code.py A3F2-8B4C-D1E9-7A5B
```

## 🔄 完整激活流程

### 用户端操作

1. 用户打开软件，点击"设置"
2. 点击"激活管理"按钮
3. 复制机器码发送给您
4. 收到激活码后，在激活窗口输入
5. 点击"立即激活"

### 您的操作

1. 收到用户的机器码
2. 运行激活码生成工具
3. 生成激活码
4. 发送给用户

---

## 💡 试用机制

- 未激活用户可免费试用 **10 次** Excel 导出
- 每次导出后显示剩余次数
- 试用用完后必须激活才能继续使用
- 激活后无限制使用

---

## ⚠️ 重要提示

1. **保密性**：请勿将 `generate_activation_code.py` 脚本分发给用户
2. **密钥一致性**：两个文件中的 SECRET_KEY 必须保持一致
   - `license_manager.py`
   - `generate_activation_code.py`
3. **机器码格式**：`XXXX-XXXX-XXXX-XXXX`（16位，用连字符分隔）

---

## 🔧 故障排除

### 激活码验证失败

**可能原因：**
- 机器码输入错误
- 密钥不一致
- 软件版本不匹配

**解决方案：**
- 让用户重新复制机器码
- 重新生成激活码
- 确认软件版本一致
