# 智能发票管理助手 (InvoiceMaster)

**智能发票管理助手** 是一款专为财务人员设计的高效发票处理工具，支持 PDF/OFC/OFD 发票的批量识别、自动排版打印、数据导出和统计分析。

![App Icon](icon_1x1_p.png)

## ✨ 核心功能

*   **⚡ 自动识别 (OCR)**
    *   支持百度 OCR 和私有 OCR 接口。
    *   自动识别发票代码、号码、金额、日期、购买方/销售方信息。
    *   支持**火车票**、**财政电子票据**等特殊票据的解析。
    *   支持批量导入和并行处理，速度极快。
*   **🖨️ 智能排版打印**
    *   **自动合并 PDF**: 将多张发票合并为一个文件进行打印。
    *   **多合一布局**: 支持 1x1 (单张), 1x2 (两张拼版), 2x2 (四张拼版) 模式。
    *   **A4/A5 自适应**: 完美适配常用纸张。
    *   **打印纠偏**: 支持强制旋转和裁剪线辅助。
*   **📊 数据管理与统计**
    *   **本地数据库**: 自动保存所有识别记录，防止数据丢失。
    *   **Excel 导出**: 一键导出标准发票台账，包含 18 个专业字段。
    *   **统计图表**: 可视化查看每月/每种类型的发票汇总。
*   **💻 跨平台支持**
    *   完美支持 Windows 10/11, Windows 7 (精简模式), macOS, 统信 UOS (Linux)。

## 🚀 快速开始

### 1. 安装与运行
*   **Windows**: 双击 `智能发票管理助手.exe`。
*   **Mac**: 双击 `智能发票管理助手.app` (首次运行可能需要在“安全性与隐私”中允许)。
*   **Linux**: 运行 `./InvoiceMaster`。

### 2. 导入发票
1.  **拖拽文件**: 直接将 PDF 发票文件拖入左侧列表区域。
2.  **自动识别**: 软件会自动OCR识别发票内容。如果某些发票金额不准确，可以**双击列表项**手动修正。

### 3. 打印设置
1.  在右侧面板选择 **打印机**。
2.  选择 **纸张类型** (A4/A5)。
3.  点击 **排版模式** 图标 (单张/两张/四张) 预览效果。
4.  点击 **"开始打印"**。

### 4. 导出 Excel
1.  点击右侧的 **"导出 Excel"** 按钮。
2.  软件会自动生成包含完整发票信息的 Excel 表格，并自动排版格式（斑马纹、重复发票高亮）。

### 5. 查看统计
1.  点击左侧顶部的 **"统计"** 按钮。
2.  查看按月份、按发票类型的金额汇总图表。

## ⚙️ 设置说明

点击主界面左上角的 **"设置"** 按钮：

*   **百度 OCR 配置**: 填入百度云的 API Key 和 Secret Key (每天有免费额度)。
*   **私有 OCR**: 如果您有自建的 paddleocr 服务，可填入接口地址。
*   **软件激活**: 输入激活码以解锁全部功能（试用版有导出次数限制）。

## ❓ 常见问题

**Q: 为什么 Win7 上界面效果不同？**
A: 为了保证流畅度，Win7 模式下会自动关闭阴影和动画特效。

**Q: 如何处理识别错误？**
A: 可以双击左侧列表中的发票项目，手动修改金额。修改后的数据会用于统计和导出。

**Q: 导出的 Excel 提示受保护？**
A: 为防止误操作，导出的 Excel 默认开启了工作表保护（密码：`InvoiceMaster2024`），您可以查看和复制，但不能修改（除非取消保护）。
