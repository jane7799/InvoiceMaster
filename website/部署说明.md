# 智能发票打印助手 - 网站部署指南

## 飞牛OS Docker 部署步骤

### 1. 准备文件
将以下文件上传到 NAS 的某个目录，例如 `/volume1/docker/invoice-web/`

```
invoice-web/
├── docker-compose.yml
└── html/
    ├── index.html
    └── logo.png
```

### 2. 上传文件方法
- 使用飞牛OS的文件管理器上传
- 或通过 SMB/NFS 共享拷贝文件

### 3. 启动容器
SSH 登录到 NAS 后执行：
```bash
cd /volume1/docker/invoice-web
docker-compose up -d
```

### 4. 访问网站
浏览器打开: `http://你的NAS-IP:8080`

### 5. 停止/重启
```bash
# 停止
docker-compose down

# 重启
docker-compose restart
```

## 常见问题

**Q: 端口 8080 被占用？**
修改 `docker-compose.yml` 中的端口，例如改为 `8088:80`

**Q: 如何更新网页？**
直接替换 `html/` 目录下的文件，无需重启容器

**Q: 如何添加下载文件？**
1. 在 `html/` 目录创建 `downloads/` 子目录
2. 将 exe/dmg/deb 文件放入其中
3. 更新 `index.html` 中的下载链接为 `downloads/xxx.exe`
