# 打包说明

## 方法一：简单打包（推荐先尝试）

```bash
# 安装 PyInstaller
pip install pyinstaller

# 打包为单个exe文件
pyinstaller --onefile --windowed --name "智能发票管理助手" InvoiceMaster.py

# 打包后的文件在 dist 目录
```

## 方法二：使用配置文件打包

```bash
# 使用 spec 文件打包
pyinstaller InvoiceMaster.spec

# 打包后的文件在 dist 目录
```

## 测试步骤

1. 在 macOS/Win10 上打包
2. 将 dist 目录下的 exe 复制到 Win7 机器
3. 测试所有功能
4. 如果失败，考虑迁移到 PyQt5

## 注意事项

- 打包后的exe文件较大（约100-200MB）
- 首次运行可能较慢
- 需要包含 license_manager.py 和二维码图片
